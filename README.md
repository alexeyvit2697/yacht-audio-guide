# Yacht Audio Guide PWA V3

PWA-версия капитанского аудиогида.

## Что умеет

- устанавливается на главный экран телефона
- работает без Telegram
- кеширует интерфейс и аудио офлайн
- отслеживает GPS
- автоматически запускает аудио при входе в радиус точки
- имеет ручной режим и тест без GPS

## Важно про автозапуск

Капитан должен нажать "Начать GPS-тур". После этого приложение может запускать треки автоматически, пока оно открыто.

## Важно про фон

PWA не гарантирует стабильную работу GPS, когда телефон заблокирован или приложение свернуто. Во время прогулки лучше держать приложение открытым.

## Как добавить аудио

Загрузи mp3 в нужную папку и проверь путь в `routes.js`.

Например:

audio: "audio/sunset/02-kalemegdan.mp3"

## Как редактировать точки

Редактируй файл `routes.js`.
