# Yacht Audio Guide

Простая капитанская панель для аудиотура на яхте.

## Как работает

- Капитан открывает приложение через Telegram Mini App
- Нажимает "Начать тур"
- Разрешает геолокацию
- Приложение показывает уведомление при приближении к точке
- Капитан вручную включает нужный аудиотрек

## Файлы

- `index.html` - основная страница
- `style.css` - внешний вид
- `app.js` - логика карты, геолокации и аудио
- `audio/` - папка для mp3-файлов

## Важно

В папке `audio/` должны лежать файлы:

- `01-intro.mp3`
- `02-kalemegdan.mp3`
- `03-confluence.mp3`
- `04-war-island.mp3`
- `05-zemun.mp3`
- `06-sunset.mp3`

Пока их нет, кнопки плеера будут видны, но звук не воспроизведется.
