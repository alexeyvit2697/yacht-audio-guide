# Yacht Audio Guide V2

Капитанская панель аудиотура для яхтенной прогулки.

## Что нового

- выбор маршрута
- отдельный файл `routes.js` для маршрутов
- отдельные папки аудио для разных маршрутов
- кнопка "Следующая точка"
- тестовый режим без GPS
- ручной запуск любого трека

## Как добавить аудио

Загрузи mp3-файлы в нужную папку.

Например, для вечернего маршрута:

- `audio/sunset/01-intro.mp3`
- `audio/sunset/02-kalemegdan.mp3`
- `audio/sunset/03-confluence.mp3`

Названия файлов должны совпадать с тем, что написано в `routes.js`.

## Как поменять точки

Открой `routes.js`.

В каждой точке можно менять:

- `title` - название точки
- `lat` - широта
- `lng` - долгота
- `radius` - радиус срабатывания в метрах
- `audio` - путь к mp3
- `note` - подсказка капитану

## Как добавить новый маршрут

В `routes.js` скопируй блок `morning` или `sunset`, поменяй ключ и данные.

Не забудь создать отдельную папку для аудио, например `audio/private/`.
